# PHP页面构建器，使用php代码生成表单表格页面

## 介绍

surface根据PHP代码快速构建出表单、表格组件页面。

## 安装

```shell
$ composer require iszsw/surface
```

## 演示

```shell
进入verdor/iszsw/surface/test目录

$ php -S localhost:888

访问
[localhost:888](http://localhost:888)

[localhost:888/helper.php](http://localhost:888/helper.php) 
```
