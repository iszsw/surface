## 特性

- 低代码
- 面向对象风格
- 自定义主题样式
- 动态生成Form页面
- 动态生成Table页面
- 丰富的表格表单样式
- 高扩展性，快速友好的扩展自定义的组件样式

## 内置功能

>  - Button
>  - Editable
>  - Expand
>  - Select
>  - Selection
>  - Switcher
>  - Writable
